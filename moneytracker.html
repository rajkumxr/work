MoneyTracker

UX CASE STUDY

The project is an innovative money tracking app designed to simplify personal finance management for users.

Roles & Resposibilities

- User Research: Competitive Analysis, User Interviews,Desk Research, User Persona, Journey Mapping

- UX Design: Information Architechture, Sketches, Low-Fidelity, Usability testing

Project Context

- July-August 2023 (2 Months)

- Solo Project

Tools Used

- Figma

- Microsoft Office

- Google Forms

Starting my journey as a UX Designer, this project was my first dive into the world of creating user experiences. I learned hands-on, applying theory to practice and gaining insights into research, ideation, and crafting interfaces that users find friendly and intuitive. It's been a foundational experience, shaping my growth in design.
Problem

Managing money has become a bit tricky, and many people struggle to keep tabs on spending and savings. Understanding our overall financial situation can be challenging. As things evolve, it's crucial to have straightforward tools to handle our money wisely.
